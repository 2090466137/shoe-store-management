## ⚡ 快速切换步骤

### 第一步：确认 Vercel 故障
```
访问：https://lhp.wang
结果：无法访问 / 502 错误 / 超时
```

### 第二步：登录 Cloudflare
```
网址：https://dash.cloudflare.com
账号：[你的 Cloudflare 账号]
```

### 第三步：修改 DNS
```
1. 点击域名：lhp.wang
2. 点击：DNS → Records
3. 找到：A 记录 @ → 216.198.79.1
4. 点击：Edit（编辑）
5. 修改为：CNAME → [你的 Netlify 域名].netlify.app
   （明天 Netlify 恢复后填写）
6. 点击：Save（保存）
```

### 第四步：等待生效
```
等待：2-5 分钟
测试：访问 https://lhp.wang
确认：网站恢复正常
```
