---

## 📱 什么是 PWA？

PWA = Progressive Web App（渐进式 Web 应用）

### 特点：
- ✅ 像 APP 一样安装到手机主屏幕
- ✅ 全屏运行，没有浏览器地址栏
- ✅ 支持离线访问
- ✅ 自动更新，无需手动下载
- ✅ 跨平台（Android + iOS）
- ✅ 无需上架应用商店

---

## 📲 安装步骤

### 方法 1：Android 手机（推荐）

#### 使用 Chrome 浏览器：

1. **打开 Chrome 浏览器**
2. **访问网址：** `https://lhp.wang`
3. **点击右上角 ⋮ 菜单**
4. **选择"添加到主屏幕"或"安装应用"**
   - 部分手机显示"安装应用"
   - 部分手机显示"添加到主屏幕"
5. **在弹出的对话框中点击"安装"**
6. **完成！** 主屏幕会出现"鞋店管理"图标 👟

#### 使用其他浏览器：
- **Edge 浏览器：** 同 Chrome
- **Firefox 浏览器：** 同 Chrome
- **小米浏览器：** 菜单 → 添加到桌面

---

### 方法 2：iPhone（iOS）

#### 使用 Safari 浏览器：

1. **打开 Safari 浏览器**
2. **访问网址：** `https://lhp.wang`
3. **点击底部中间的"分享"按钮** 📤
4. **向下滚动，找到"添加到主屏幕"**
5. **点击"添加到主屏幕"**
6. **输入名称：** 鞋店管理（或保持默认）
7. **点击右上角"添加"**
8. **完成！** 主屏幕会出现图标 👟

**注意：** iOS 必须使用 Safari 浏览器，Chrome 不支持！

---

### 方法 3：电脑（Windows/Mac）

#### 使用 Chrome 浏览器：

1. **打开 Chrome 浏览器**
2. **访问网址：** `https://lhp.wang`
3. **点击地址栏右侧的"安装"图标** ⊕
   - 或点击右上角 ⋮ → 更多工具 → 创建快捷方式
4. **勾选"在窗口中打开"**
5. **点击"安装"**
6. **完成！** 桌面会出现快捷方式

## 🌐 离线功能

### 已支持的离线功能：
- ✅ 查看商品列表
- ✅ 查看销售记录
- ✅ 查看会员信息
- ✅ 查看统计数据

### 需要网络的功能：
- ⚠️ 添加/编辑/删除数据（需要同步到云端）
- ⚠️ 首次登录（需要验证账号）

### 工作原理：
1. 首次访问时，系统会缓存页面和数据
2. 离线时，可以查看已缓存的数据
3. 恢复网络后，自动同步到云端

---

## 🔄 更新机制

### 自动更新：
- ✅ 每次打开 APP 时自动检查更新
- ✅ 发现新版本时自动下载
- ✅ 下次启动时自动应用更新
- ✅ 无需手动下载安装

### 强制更新：
如果需要立即更新：
1. 关闭 APP
2. 重新打开
3. 新版本自动生效

## 🗑️ 卸载方法

### Android 手机：
1. **长按"鞋店管理"图标**
2. **选择"卸载"或拖到"卸载"区域**
3. **确认卸载**

或者：
1. **设置 → 应用管理**
2. **找到"鞋店管理"**
3. **点击"卸载"**

### iPhone：
1. **长按"鞋店管理"图标**
2. **点击"删除 App"**
3. **确认删除**

### 电脑：
1. **右键点击桌面快捷方式**
2. **选择"删除"**

### PWA 配置：
```json
{
  "name": "鞋店仓库管理系统",
  "short_name": "鞋店管理",
  "display": "standalone",
  "theme_color": "#1989fa",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192" },
    { "src": "/icon-512.png", "sizes": "512x512" }
  ]
}
```

### Service Worker：
- 缓存策略：Cache First（优先使用缓存）
- 离线支持：自动缓存页面和数据
- 自动更新：每次启动检查新版本
