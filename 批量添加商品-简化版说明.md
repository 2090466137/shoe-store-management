# 批量添加商品 - 简化版功能说明

## 🎯 设计理念

**简单、快速、直观** - 让批量添加商品像点外卖一样简单！

## ✨ 核心优化

### 📦 统一库存模式（适合数量一致）
保持原有简单设置，一键搞定

### 🎨 单独设置模式（全新简化）

#### 1️⃣ **卡片式显示 - 一目了然**
```
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│38码 │ │39码 │ │40码 │ │41码 │
│10件 │ │15件 │ │20件 │ │15件 │
└─────┘ └─────┘ └─────┘ └─────┘
```
- 渐变色卡片，美观醒目
- 显示尺码和当前库存
- **点击即可修改**

#### 2️⃣ **快速输入 - 秒速设置**
点击任意尺码卡片 → 弹出输入框 → 输入数量 → 完成！

**输入框特点：**
- 大号字体，清晰易读
- 自动聚焦并选中
- 常用数量快捷按钮（5、10、15、20）
- 一键点击快速填入

#### 3️⃣ **两个神器按钮**

**🔹 快速批量**
- 一次性设置所有尺码
- 弹窗输入，带常用数量快捷按钮
- 适合大部分尺码数量相同的情况

**🔹 智能填充**
- 自动按热门尺码分配数量
- 中间尺码多（如40-42码）
- 两端尺码少（如38、44码）
- 符合实际销售规律

## 📱 使用流程

### 场景 1：所有尺码数量相同
**操作步骤**：
1. 选择"统一库存"模式
2. 输入默认库存：10
3. 完成！✅

**耗时**：5秒

---

### 场景 2：大部分相同，个别不同
**示例**：38-44码，大部分10件，41码20件

**操作步骤**：
1. 选择"单独设置"模式
2. 点击"快速批量"按钮
3. 输入10（或点击快捷按钮）
4. 点击"41码"卡片
5. 修改为20
6. 完成！✅

**耗时**：10秒

---

### 场景 3：每个尺码数量不同
**示例**：
- 38码：5件
- 39码：8件
- 40码：15件
- 41码：20件
- 42码：15件
- 43码：8件
- 44码：5件

**操作步骤**：
1. 选择"单独设置"模式
2. 依次点击每个尺码卡片
3. 输入对应数量（可用快捷按钮）
4. 完成！✅

**耗时**：30秒

---

### 场景 4：按热门尺码分配
**操作步骤**：
1. 选择"单独设置"模式
2. 点击"智能填充"按钮
3. 系统自动分配：
   - 38码：7件
   - 39码：12件
   - 40码：17件
   - 41码：20件（中间最多）
   - 42码：17件
   - 43码：12件
   - 44码：7件
4. 微调个别尺码（如需要）
5. 完成！✅

**耗时**：5秒 + 微调时间

## 💡 智能填充算法

**原理**：使用抛物线函数分配库存

```
库存 = 5 + (1 - (2×位置 - 1)²) × 15
```

**效果**：
- 最小值：5件（两端）
- 最大值：20件（中间）
- 平滑过渡

**示例**（7个尺码）：
```
位置:  0    1    2    3    4    5    6
尺码: 38   39   40   41   42   43   44
库存:  7   12   17   20   17   12    7
```

## 📝 使用建议

1. **数量完全相同** → 用"统一库存"
